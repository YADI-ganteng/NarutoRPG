Game RPG Open World dengan 53 karakter dari Naruto!

## 🎮 Fitur
- 🌍 Open World 4000x3000 pixel
- 👤 53 Karakter Playable
- ⚔️ Battle System (Attack, Skill, Ultimate)
- 🎵 630 Sound Effects & BGM
- 🗺️ Minimap Radar
- 📜 Quest System
- 🤝 NPC Interaction
- 💰 Gold & Level System
- 🕹️ Joystick + Keyboard Support
- 📱 Mobile Friendly

## 🎯 Controls
- 🕹️ **Joystick kiri** / **WASD** = Move
- 👊 **Tombol merah** / **J** = Attack
- ⚡ **Tombol biru** / **K** = Skill
- 🌟 **Tombol kuning** / **L** = Ultimate
- 🤝 **Tombol hijau** / **E** = Interact NPC
